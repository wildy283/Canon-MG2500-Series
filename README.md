# Canon-MG2500-Series-Driver

##### Untuk scangearmp wajib install packages libusb-0.1-4
##### untuk installasinya silakan copy command di bawah

# Ubuntu / Debian
`apt install libusb-0.1-4 libusb-0.1-4:i386 libpango1.0-0 -y`

# Rpm
`cari sendiri nama packagesnya !`

# Info
untuk scan bisa pakai software simple-scan

atau bisa di install pakai command ini jika di debian/ubuntu

`apt install simple-scan -y`


# Additional

`lpadmin -p MG2500 -o print-color-mode-default=color`
